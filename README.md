# Swiss Digital Pact - Website

This is the Next.js implementation of the Swiss Digital Pact website, a federal popular initiative for digital sovereignty and security in Switzerland.

## Technology Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **Design**: Premium Swiss-inspired aesthetics with glassmorphism and Swiss Red (#E30613)

## Features
- **Responsive Navigation**: Full desktop menu and mobile sidebar.
- **SEO Optimized**: Dynamic metadata for each page, semantic HTML5 structure.
- **Multi-language Ready**: Pre-architected header with language switcher.
- **Direct Action Focused**: High-visibility "Sign the Initiative" CTAs throughout the site.

## Directory Structure
- `/src/app`: Application routes and layouts.
- `/src/components`: Reusable UI components (Header, Footer).
- `/public`: Static assets (Logos, PDFs).

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Page Overview
- **Home**: Hero section with mission statement and CTA.
- **The Initiative**: Detailed breakdown of the constitutional proposal.
- **Sign**: Step-by-step instructions on signing and mailing the form.
- **Support**: Donation options (IBAN, Platforms, QR).
- **Contact**: Multi-channel contact options and inquiry form.
